# 霍纳规则(秦九韶算法)
## 直接迭代进行多项式求值
```python
sum = 0
for k in range(0, n + 1):
    sum = sum + a_k * x^k
```
对于每一个k，计算复杂度为k + 2次

T(n) = an + b(2 + 3 + ... + (n =+ 2)) = an + b(n(n + 4) / 2)

这会引入过多的乘法，最终导致算法的运行时间为Θ(n^2)

## 霍纳规则计算多项式
```python
y = 0
i = n
while i >= 0:
    y = a_i + x * y
```
采取比较少的乘法运算，大大提高计算效率

对于每一个k，计算复杂度为2

T(n) = an + b * 2 * n = (a + 2b)n

即霍纳规则算法的运行时间为Θ(n)
